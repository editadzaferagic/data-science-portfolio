# Spotify Data Analysis

This project explores a dataset of over 10,000 songs to identify trends in music popularity and artist performance.

## What I did:

* Loaded and explored dataset
* Performed exploratory data analysis (EDA)
* Identified top artists and most popular songs
* Analyzed trends in explicit content over time
* Created visualizations to present insights

## Tools:

* Python
* pandas
* matplotlib
* seaborn
* Google Colab
