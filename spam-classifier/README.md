# Spam Message Classifier (NLP)

This project implements a machine learning-based spam detection tool using natural language processing techniques.

---

## Project Overview

* Loaded and cleaned text message data
* Converted text into numerical features using TF-IDF vectorization
* Trained a Logistic Regression model for spam classification
* Built an interactive console application for real-time message prediction

---

## Technologies Used

* Python
* scikit-learn
* pandas
* NLP (TF-IDF Vectorization)

---

## How to Run

1. Run the script:

```
python spam_classifier.py
```

2. Enter a message in the console

3. The model will classify it as **SPAM** or **HAM**

4. Type `exit` to stop the program

* Real-time spam detection

* End-to-end ML workflow (data → model → prediction)
