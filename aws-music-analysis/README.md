# AWS Music Data Analysis

This project demonstrates a complete cloud-based data analysis workflow using AWS S3 and Python.

The goal was to process and analyze music chart data from multiple sources and generate aggregated insights about artist performance.

---

## Project Overview

* Uploaded and organized multiple CSV datasets in AWS S3
* Programmatically accessed cloud data using Python (Boto3)
* Combined and processed data from multiple files
* Performed aggregation and analysis to identify top-performing artists
* Generated a final dataset of top 20 artists and stored results back in S3

This project simulates a real-world cloud data engineering and analytics workflow.
